# Articlos Marketing Site

Static HTML/CSS version ready for GitHub Pages.

## Deploy on GitHub Pages

1. Upload `index.html` and `styles.css` to the root of your repository.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Save.
5. Wait a minute and open your Pages URL.
